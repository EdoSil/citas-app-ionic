# App de Citas – Ionic

Este proyecto corresponde a una aplicación móvil híbrida desarrollada con Ionic y Angular, creada como parte de la **Evaluación 2 del ramo Programación Híbrida**. La aplicación permite gestionar citas mediante una estructura modular, utilizando componentes reutilizables y servicios para la lógica de negocio.

El proyecto considera el uso de SQLite para la persistencia de datos en dispositivos móviles. Debido a las limitaciones del entorno web, la persistencia fue simulada mediante servicios y almacenamiento local, dejando preparada la arquitectura para su ejecución mediante Capacitor.

## Tecnologías
- Ionic
- Angular
- TypeScript
- Capacitor
- SQLite
